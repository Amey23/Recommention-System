# Recommention-System
**This repository contains the code for building movie recommendation system.**

•	Approach 1 :- By weighted average
•	Approach 2 :- By popularity and genre
•	Approach 3 :- By content(overview) based

2. pandas
3. numpy
4. scipy
5. scikit-learn
6. scikit-surprise
7. matplotlib
8. seaborn
9. jupyter notebook
